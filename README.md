TacTics: Web-Based Learning Media for Football Strategy Management
TacTics เป็นเว็บแแอปพลิเคชันสื่อการสอนที่ออกแบบมาเพื่อช่วยให้นักเรียนหรือผู้ที่สนใจได้เรียนรู้และทำความเข้าใจเกี่ยวกับกลยุทธ์การวางแผนฟุตบอลผ่านระบบ Interactive และการทำแบบฝึกหัดเสมือนจริง

📑 รายละเอียดโปรเจกต์ (Overview)
โปรเจกต์นี้พัฒนาขึ้นเพื่อแก้ปัญหาการเรียนรู้เรื่องแผนการเล่นฟุตบอลที่เข้าใจยาก โดยเปลี่ยนจากภาพนิ่งในตำราให้กลายเป็นระบบที่ผู้ใช้สามารถทดลองจัดวางตัวผู้เล่นได้ด้วยตนเอง มีระบบจัดการข้อมูลผู้ใช้ และระบบตรวจสอบคะแนนจากการทำแบบฝึกหัด

✨ คุณสมบัติเด่น (Features)
User Management: ระบบสมัครสมาชิกและล็อกอินผ่าน Firebase Authentication

Personal Profile: ผู้ใช้สามารถดูและแก้ไขข้อมูลส่วนตัว (ชื่อ, รหัสประจำตัว, สาขาวิชา, ชั้นเรียน) ได้เอง

Admin Dashboard: มีหน้าสำหรับ Admin เพื่อดูรายชื่อนักศึกษาและข้อมูลทั้งหมดในระบบ

Interactive Exercises: แบบฝึกหัดการจัดแผนการเล่นฟุตบอล (Drag & Drop) เช่น แผน 4-4-2 และ 3-5-2

Real-time Database: ใช้ Firebase Firestore ในการจัดเก็บและดึงข้อมูลแบบเรียลไทม์

Responsive Design: รองรับการแสดงผลทั้งบนคอมพิวเตอร์และสมาร์ทโฟนด้วย Bootstrap 5

🛠 เทคโนโลยีที่ใช้ (Tech Stack)
Frontend: HTML5, CSS3 (Google Fonts - Prompt), JavaScript (ES6+)

UI Framework: Bootstrap 5.3

Backend & Database: Firebase

Firestore Database (สำหรับเก็บข้อมูลผู้ใช้)

Firebase Authentication (สำหรับระบบ Login/Logout)

Library: Firebase SDK 10.8.0
